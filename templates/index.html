Classify Between Chhau and Kathakali, Two Classical Dance Forms of India Using Artificial Intelligence

by Ritobrata Ghosh







How this works:
Just upload an image of Chhau or Kathakali dance using the "Browse..." button and click on "Predict" to get prediction of the dance form.

This web app depends on a batch-learning model leveraging Convolutional Neural Network and transfer learning. The model is trained Using PyTorch and fastai v1 libraries. This is an Open Source project. Visit GitHub Repository for code.
